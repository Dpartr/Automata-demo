Conway's Game of Life

Interval Speed

This is a Demo of Conway's game of life. The rules are. If a cell is dead and it has exactly 3 live neighbors, it will change state to alive. If a cell is alive and has 2 or 3 neighbors, it will continue to live, otherwise dying as if by under or over population Press Generate Noise to Randomly change cell states. Press Generate Seed to use the default start for the game of life.

If you are viewing this on a desktop browser, hover your mouse over the grid to interactively change cell states.
