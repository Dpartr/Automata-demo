Conway's Game of Life

This is a Demo of Conway's game of life. The rules are. If a cell is dead and it has exactly 3 live neighbors, it will change state to alive. If a cell is alive and has 2 or 3 neighbors, it will continue to live, otherwise dying as if by isolation or overcrowding. Press Generate Noise to Randomly change cell states. Press Place Boomerang to create a boomerang in the center.

If you are viewing this on a desktop browser, hover your mouse over the grid to interactively change cell states.
